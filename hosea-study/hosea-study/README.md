# 호세아서 상세 연구본

일곱 종의 호세아서 주석을 종합한 한국어 연구 노트. 배경 연구부터 장절 주해, 원어 풀이,
설교자를 위한 신학적 메시지, 쟁점 대조표까지 한 문서에 담았다.
의존성 없는 단일 HTML 파일이라 브라우저만 있으면 어디서나 열린다.

---

## 보기

**로컬에서** — `index.html`을 브라우저로 열면 된다. 서버가 필요 없다.

**GitHub Pages로 공개하려면** — 저장소 `Settings` → `Pages` → *Build and deployment*의
Source를 **Deploy from a branch**, 브랜치를 `main` / `(root)`으로 지정하면
`https://<사용자명>.github.io/<저장소명>/` 에서 바로 열린다. 별도 워크플로가 필요 없다.

> 본문 서체는 Google Fonts(Noto Serif KR·Noto Serif Hebrew)를 불러온다.
> 오프라인에서는 시스템 명조로 대체되며 레이아웃은 그대로 유지된다.

## 편집하기

`index.html`은 **빌드 산출물이다. 직접 고치지 말 것.**
내용은 `src/`의 조각 파일에서 고치고 다시 빌드한다.

```bash
python3 build.py           # src/ → index.html 생성 + 검증
python3 build.py --check   # 파일을 쓰지 않고 검증만 (CI용)
```

빌드 스크립트는 표준 라이브러리만 쓰며, 목차 앵커 연결·태그 균형·섹션 개수를 검사한 뒤
통과할 때만 파일을 쓴다.

### `src/` 구조

| 파일 | 역할 |
|---|---|
| `_head.html` | DOCTYPE, `<head>`(스타일 전체), 사이드바 목차, 히어로 |
| `00-overview.html` … `08-bibliography.html` | 각 부의 `<section>` |
| `_foot.html` | 푸터와 닫는 태그 |

부의 순서와 구분선 라벨은 `build.py`의 `PARTS` 목록이 관리한다.
새 부를 추가하려면 `src/`에 파일을 넣고 이 목록에 한 줄 더하면 된다.
목차 링크는 `_head.html`의 `<nav class="toc">`에서 함께 고친다.

### 본문 작성 규칙

문서 전체가 쓰는 클래스는 다음과 같다.

| 클래스 | 용도 |
|---|---|
| `.chap` / `.chip-num` | 장 블록과 장 번호 배지 |
| `.vs` / `.ref` | 절 단위 주해 블록과 그 절 표시 |
| `.hw` | 히브리어 인라인 (RTL 처리됨) |
| `.c k` `.c g` `.c hb` `.c d` `.c kd` `.c s` `.c m` | 주석가 칩 |
| `.note` | 보충 설명 (녹색) |
| `.caveat` | 주석이 갈리는 지점·주의 사항 (와인색) |
| `.gold-box` | 핵심 인용·강조 (금색) |
| `.lead` | 장 도입 요약 |
| `.back` | 각 부 끝의 "↑ 처음으로" |

히브리어는 `<span class="hw">…</span>`로 감싸고 괄호 안에 (발음, 뜻)을 병기한다.
해석 진술에는 출처 칩을 단다.

### 주석가 칩

| 칩 | 주석 | 해석 축 |
|---|---|---|
| **K** | Kidner, BST (1981) | 경건·문학·수사, 간결한 강해 |
| **G** | Goldingay, Baker (2021) | 히브리어 정밀·번역, 신학적 성찰 |
| **Hb** | Hubbard, TOTC (1989) | 복음적·목회적, 양식·주제 종합 |
| **D** | Dearman, NICOT (2010) | 고대근동·주해 종합, 가정 근원은유 |
| **KD** | Keil–Delitzsch (19c) | 고전 문법·정통 |
| **S** | Stuart, WBC (1987) | 언약 저주·축복 틀, 오경 의존 |
| **M** | Moon, Apollos (2018) | 신학·수용사, 단일저작 옹호 |

## 디자인

느헤미야 연구본과 구별되도록 석류(와인) · 곡식(금) · 포도나무(녹색) 색조로 짰다.
각 부 사이의 구분선은 이스르엘(*"하나님이 씨 뿌리신다"*)을 형상화한 **파종 고랑** 문양이다.
인쇄용 스타일과 좁은 화면 대응이 포함되어 있다.

## 출처와 저작권

이 문서는 일곱 종 주석의 논지를 **한국어로 요약·재구성**한 개인 연구 노트다.
성경 인용은 개역개정을 따른다. 공개 저장소에 올리기 전에 반드시 [`NOTICE.md`](NOTICE.md)를 읽을 것.

서지의 출판 정보와 각 주석의 세부 논지는 요약이므로, 인용·발표 시에는 해당 면수를
원문에서 직접 확인해야 한다.
